what is document object model dom...?

ans: The Document Object Model (DOM) is a tree-like structure that represents an HTML or XML document, allowing you to access, modify, and interact with elements on a webpage using programming languages like JavaScript.

how do you select an element from the dom...?

ans: getElementById()


what is event delegatetion in the contex of dom and whay is it useful...?

ans: Event delegation is attaching one event listener to a parent to manage events for its child elements via event bubbling.
Why useful:
Fewer listeners: Better performance. Dynamic elements: Handles new child elements. Simpler code: Centralized event handling.


how do you manipulate an elements attributes and styles the dom...?

ans: Set attribute: element.setAttribute('attribute', 'value'); Get attribute: element.getAttribute('attribute'); Remove attribute: element.removeAttribute('attribute');

Set style: element.style.property = 'value';
